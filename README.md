# 📘 Git Commands Cheat Sheet

- `git init` → Initializes a new Git repository  
- `git config --global user.name "Your Name"` → Sets Git username  
- `git config --global user.email "your@email.com"` → Sets Git email  

- `git status` → Shows current file status (tracked/untracked/changes)  

- `git add .` → Adds all files to staging area  
- `git commit -m "message"` → Saves changes to Git history  

- `git rm <file>` → Deletes a file and stages the deletion  

- `git branch feature-login` → Creates a new branch  
- `git checkout <branch>` → Switches to another branch  
- `git branch -m main` → Renames current branch to main  

- `git merge feature-login` → Merges a branch into current branch  

- `git log` → Detailed commit history  
- `git log --oneline` → Short and readable history  
- `git diff` → Shows exact code changes  

- `git push` → Sends local commits to GitHub  
- `git pull` → Gets latest changes from GitHub  

- `.gitignore` → File that tells Git what to ignore  

- `git checkout HEAD~1 students.txt` → Restores file from previous commit  

- `git remote add origin <url>` → Connects local repo to GitHub  

- `git clone <repo-url>` → Copies a repository from GitHub  

- `git fork` → Copies another user’s repo on GitHub (done via website)
